# Jack Hole Jackery Studios: CALEB Vanguard
## Executive Orchestration Layer (v2.0)

CALEB (Creative Automation & Local Engineering Builder) is a 3-agent autonomous developer optimized for **Intel Arc A770** hardware.

### Current Features
- **3-Agent Logic:** Planner, Coder, and Verifier roles for high-precision generation.
- **Hardware Awareness:** Live VRAM/RAM load balancing and safety lockouts.
- **Template Arsenal:** Rapid scaffolding using `.tpl` boilerplates.
- **XML Toolchain:** Secure filesystem and terminal execution.

### Tech Stack
- **Inference:** Ollama (Qwen2.5-Coder:7b)
- **UI:** PyQt5 (Dark Mode / Hybrid Tabbed View)
- **Hardware Tracking:** PyOpenVINO / PSUtil

### Project Mission
To orchestrate the modular "Jack Hole Jackery Studios" creative suite, spanning Image Forge, Video Motion VFX, and Audio Lab.
